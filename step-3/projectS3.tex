\section{Objectives}
The goal of this step in the project is to integrate the 4-bit ALU components from Step 2 into a control unit that selects the appropriate operation based on a 4-bit opcode. This creates a complete ALU that can perform arithmetic and logic operations on 4-bit inputs.

\section{ALU Control Unit Design}
The control unit takes a 4-bit opcode and two 4-bit inputs A and B, and produces an 8-bit output based on the selected operation. It instantiates all the 4-bit modules from Step 2 and uses a case statement to select the output.

\subsection{Operation Codes}
The following table lists the supported operations and their corresponding opcodes:

\begin{itemize}
\item 4'b0000 (0)  - ADD   : out[4:0]  = \{Cout, A+B\}
\item 4'b0001 (1)  - SUB   : out[4:0]  = \{Bout, A-B\}
\item 4'b0010 (2)  - MUL   : out[7:0]  = \{upper4, lower4\} of A*B
\item 4'b0011 (3)  - DIV   : out[7:0]  = \{remainder, quotient\} of A/B
\item 4'b0100 (4)  - AND   : out[3:0]  = A \& B
\item 4'b0101 (5)  - OR    : out[3:0]  = A | B
\item 4'b0110 (6)  - XOR   : out[3:0]  = A \^{} B
\item 4'b0111 (7)  - SHIFT : out[7:4]  = bits shifted out, out[3:0] = shifted result \\
                          B[3]=dir(0=L,1=R), B[2:1]=amount(0-3), B[0]=fill
\item 4'b1100 (12) - NAND  : out[3:0]  = \~(A \& B)
\item 4'b1101 (13) - NOR   : out[3:0]  = \~(A | B)
\item 4'b1110 (14) - XNOR  : out[3:0]  = \~(A \^{} B)
\item 4'b1111 (15) - NOT   : out[7:4]  = \~B,  out[3:0] = \~A
\end{itemize}
